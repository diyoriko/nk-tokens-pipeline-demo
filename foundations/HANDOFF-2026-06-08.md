## Gotchas
- Local repo `main` can lag origin — `git fetch && git merge --ff-only origin/main` before reviewing "the source of truth".
- Build CSS (`build/`) is gitignored; only `tokens/tokens.json` is committed.
- Figma MCP is desktop-bound (only sees the open file's near-viewport); REST/PAT reads whole shared files. Project file listing needs a work-account PAT with `projects:read`.
- Product Figma files are 300MB+ illustration-heavy — full fill-sweep impractical; use the code extracts for product colour usage.
- No `Co-Authored-By` in commits (account trust recovery).
